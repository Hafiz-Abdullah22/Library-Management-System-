# Library-Management-System-
A Library Management System to manage books, users, and transactions.

# Project-Goals
Goal 1: Develop a console-based system for library management.
Goal 2: Implement classes to represent books, users, and transactions.
Goal 3: Allow users to check out and return books, with due dates and late fees.
Goal 4: Provide functionality for adding and removing books and users.
