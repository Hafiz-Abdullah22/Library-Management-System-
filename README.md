# LIBRARY MANAGEMENT SYSTEM

## Project Title  
**Library Management System (C++)**

## Objective  
To create an efficient and user-friendly platform for managing library resources using C++.  
The system will emphasize:  
- File handling for managing book records.  
- Robust exception handling for seamless operation.

## Features  
1. **Book Management**  
   - Add, view, and delete book records using file handling.  
   - Ensure structured and organized data storage.  

2. **Exception Handling**  
   - Graceful handling of invalid user inputs to avoid system crashes.  
   - Management of missing or corrupt files to maintain data integrity.
